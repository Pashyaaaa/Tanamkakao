# TanamKakao 🌿

TanamKakao adalah platform digital yang dirancang untuk memodernisasi cara petani kakao mengelola peremajaan kebun dan penanaman. Kami percaya bahwa teknologi harus memudahkan, bukan membingungkan khususnya untuk para Petani.

### 🚀 Fitur Utama
Platform ini dibangun untuk membantu petani dari nol hingga panen:

* **🌱 Smart Seed Matching**: Algoritma sederhana untuk mencocokkan jenis tanah dengan varietas bibit kakao terbaik.
* **📊 Kebun Saya**: Ruang personal untuk memantau data kebun, estimasi biaya, dan melacak pertumbuhan tanaman.
* **📅 Jadwal Rawat TBM Dinamis**: Kalender digital interaktif agar tidak lupa jadwal pemupukan dan perawatan tanaman muda (TBM).
* **📍 Direktori Penjual**: Sistem CRUD untuk menyimpan daftar penyedia bibit terpercaya di sekitar Anda.

---

### 🛠️ Tech Stack
Dibuat dengan pendekatan *Clean & Vanilla* agar ringan dan mudah dimodifikasi:

- **Frontend**: HTML5, CSS3 (Custom Styling)
- **Logic**: Vanilla JavaScript (ES6+)
- **Architecture**: Modular Single Page Application (SPA)

---

### 📦 Cara Menjalankan
Tidak perlu instalasi yang rumit, cukup jalankan secara lokal:

1. **Clone repository ini**:
   `git clone https://github.com/username/tanamkakao.git`

2. **Buka folder**:
   `cd tanamkakao`

3. **Jalankan**:
   Cukup buka file `index.html` di browser favorit kamu.
